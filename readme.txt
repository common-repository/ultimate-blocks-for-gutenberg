== Description ==

[Master Blocks](https://jeweltheme.com/master-blocks) is a powerful WordPress Gutenberg blocks plugin that helps you to create beautiful pages in default WordPress Editor. It extends the capabilities of the Gutenberg editor, providing a rich collection of blocks for building stunning web pages without the need for coding.

> [Try live demo](https://masterblocks.jeweltheme.com/) | [Contact Us](https://jeweltheme.com/contact) | [Join Facebook Community](https://www.facebook.com/groups/1998722457117243)

Master Blocks provides 20+ blocks and 5+ extensions for the Gutenberg editor, it helps you to bring your concept into reality.

**Gutenberg Blocks:**

- **[Heading & Sub Heading](https://masterblockstemplates.jeweltheme.com/demos/heading-and-subheading/):** Organize unique title section with Heading & Sub heading block.
- **[Hero Image](https://masterblockstemplates.jeweltheme.com/demos/hero-image/):** Create hero sections with background images, title, button.
- **[Counter Up](https://masterblockstemplates.jeweltheme.com/demos/counter-up/):** Display animated counters for statistics and achievements.
- **[Click to Tweet](https://masterblockstemplates.jeweltheme.com/demos/click-to-tweet/):** Encourage social sharing with easy-to-use tweetable content.
- **[Feature Box](https://masterblockstemplates.jeweltheme.com/demos/feature-box/):** Showcase key features or services attractively.
- **[Social Share](https://masterblockstemplates.jeweltheme.com/demos/social-share/):** Increase social engagement with customizable share buttons.

- **Transform:** Use all CSS transform properties to create creative sections and blocks.
- **Animation:** Add animations to any block or sections in gutenberg editor.
- **Positioning:** Control the positioning of your blocks within your content.

Master Blocks is constantly evolving, with regular updates and new features to enhance your WordPress editing experience. Give it a try today and unlock the full potential of the Gutenberg editor!

== Installation ==

1. Upload the 'ultimate-blocks-for-gutenberg' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit the 'Gutenberg' editor to start using Master Blocks' powerful blocks.

= After activation =
1. Visit Menu **Master Blocks > Settings** menu.
2. Enable the individual Blocks you would like to use.
